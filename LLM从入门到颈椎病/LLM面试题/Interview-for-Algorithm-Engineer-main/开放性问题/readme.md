《三年面试五年模拟》之开放性问题高频考点
--------------------

**Rocky从工业界、应用界、竞赛界以及学术界角度出发**，总结归纳AI行业的一些开放性本质问题，**这些本质问题不仅能够用于面试官在面试时的提问，也可以用作面试者对面试官的提问**，在面试的最后阶段让大家进入更深刻本质的探讨与交流。
