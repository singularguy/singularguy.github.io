AIOS：人工智能代理操作系统

AIOS 是人工智能代理操作系统，它将大型语言模型 (LLM) 嵌入到操作系统中，并促进基于 LLM 的人工智能代理的开发和部署。AIOS 旨在解决基于 LLM 的代理在开发和部署过程中遇到的问题（例如，调度、上下文切换、内存管理、存储管理、工具管理、代理 SDK 管理等），从而为代理开发人员和代理用户提供更好的 AIOS-Agent 生态系统。AIOS 包括 AIOS 内核（此AIOS存储库）和 AIOS SDK（Cerebrum存储库）。AIOS 支持 Web UI 和终端 UI。
